Projects

  • Liquid Legends (Unity Original Game)

Inspiration

We wanted to see how good of a game we could make in 24 hours.

What it does

It's a small indie game that we hope will charm players.

How we built it

The backend was built in C# through Unity and assets were designed in Piskel.

Challenges we ran into

We encountered some hitbox and collision issues with the characters.

Accomplishments that we're proud of

Creating an original soundtrack for the game.

What we learned

How to integrate AI in video game development.

What's next for Liquid Legends

We plan to host it on Steam.
